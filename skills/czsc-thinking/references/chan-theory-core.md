# 缠论核心理论

本文档整理了缠中说禅（2006-2008年）原文思想中的核心理论内容，供深入学习和参考。

## 核心理念

### 1. 不测而测

```
技术分析的最终意义不是去预测市场要干什么，而是市场正在干什么，
是一种当下的直观。在市场上所有的错误都是离开了这当下的直观，
用想象、用情绪来代替。
```

**关键要点：**
- 不预测市场未来会怎样
- 关注市场当下正在发生什么
- 根据走势本身的规律来操作
- 避免被情绪和想象误导

### 2. 完全分类

```
对于一个真正的操作者，没有任何情况是意外的，因为，所有的情况
都被完全地分类了，所有相应的对策都事先有了，只是等着市场自己
去选择，去触及我们事先给定的开关。
```

**关键要点：**
- 事先完成所有可能情况的分类
- 为每种情况准备好应对策略
- 不依赖预测，只等市场选择
- 机械化执行预定策略

### 3. 级别思维

```
学了缠论的理论，脑子里必须时刻有两个字：级别。如果连级别都搞
不清楚，你还419？被419还差不多。有了级别，就是节奏问题了，419，
就是见好要收，而不是天长地久。
```

**关键要点：**
- 任何分析都必须明确级别
- 不同级别有不同的操作策略
- 级别决定持仓时间和预期收益
- 混淆级别是致命错误

## 操作哲学

### 市场本质

**市场是人心的合力：**
```
走势反映的是人的贪嗔痴疑慢，如果你能通过走势当下的呈现，而观
照其中参与的心理呈现，就等于看穿了市场参与者的内心。
```

**关键认知：**
- 市场走势是所有参与者心理的合力
- 贪婪、恐惧、愚蠢是永恒的人性
- 技术分析就是解读人性在图形上的投射
- 了解多空双方的心理状态

### 风险控制

**生存第一：**
```
市场的第一原则就是生存，只要你30年后还能活下来，自然就是最大
的牛人。整体的失败是一次都不能发生的，只要发生一次，基本就翻
不了身了。
```

**风险原则：**
- 永远不要让一次失败毁掉所有
- 控制仓位，分散风险
- 严格止损，不抱幻想
- 长期生存比短期暴利重要

### 交易纪律

**机械操作：**
```
市场就如同一头牛，只有目无全牛，才可能随心解之而合其关节。机
械化操作的本质就是目无全牛而合其关节，因为，根据缠论理论，市
场的结构已经被彻底分解。
```

**纪律要求：**
- 严格按照系统操作，不加入个人情绪
- 市场已被理论完全分解，只需执行
- 挣钱是良好习惯和策略的结果
- 远离聪明，坚持机械操作

## 分析方法

### 底部分析

**底部定义：**
```
第一类买点出现后，直到该买点所引发的中枢第一次走出第三类买卖
点前，都可以看成底部构造的过程。不是在底部的区间上买，而是相
反，应该和中枢震荡的操作一样，在区间下探失败时买，这才是最好
的买点。
```

**分析步骤：**
1. 识别第一类买点（下跌后不创新低）
2. 观察中枢形成过程
3. 等待向下试探失败
4. 在下探失败后买入
5. 等待第三类买点确认底部完成

### 买卖点判断

**三类买卖点：**

**第一类买点：**
- 下跌趋势后，向上笔不创新低
- 标志着下跌动能衰竭
- 风险最大，收益也最大

**第二类买点：**
- 一买后回调不破前低
- 确认底部有效性
- 风险收益比最佳

**第三类买点：**
- 向上离开中枢的回调点
- 趋势确立后的参与机会
- 最安全但收益相对较小

### 节奏把握

**操作节奏：**
```
操作其实很简单，一个基本的原则就是，任何走势，无论怎么折腾，
都逃不出这个节奏，就是底、顶以及连接两者的中间过程。在两头的
操作节奏就是中枢震荡，只是底的时候要先买后卖，顶的时候要先卖
后买。
```

**三阶段操作：**
1. **底部阶段**：中枢震荡，先买后卖
2. **上涨阶段**：持有为主，适当高抛低吸
3. **顶部阶段**：中枢震荡，先卖后买

## 心理修炼

### 克服贪婪

**你的喜好，就是你的死亡陷阱：**
```
在市场中要生存，第一条就是在市场中要杜绝一切喜好。市场的诱惑，
永远就是通过你的喜好而陷你于死亡。市场中需要的是露水之缘而不
是比翼之情。
```

**修炼要点：**
- 不要爱上任何股票
- 股票只是赚钱的工具
- 见好就收，不贪天长地久
- 赚钱才是唯一目标

### 克服恐惧

**市场陷阱的利用：**
```
当你要买的时候，空头的陷阱就是你的最佳机会，当你要卖的时候，
多头的陷阱当然就是你的天堂。
```

**利用恐惧：**
- 别人恐惧时我贪婪
- 空头陷阱是买入良机
- 多头陷阱是卖出时机
- 市场永远有人在错误的位置操作

### 克服愚蠢

**认清自己：**
```
要认清市场，首先要认清自己，知道自己的弱点在哪里，自己在市场
中的每个行为，都要清楚地意识到。每天收盘后，都找十分钟，把自
己当天的操作以及看盘时的心理过程复一次盘。
```

**自我认知：**
- 每日复盘检讨
- 了解自己的弱点
- 意识到每个操作的心理动机
- 不断修正错误模式

## 常见误区

### 预测陷阱

**错误思维：**
- 试图预测顶部在哪里
- 猜测底部何时出现
- 幻想未来会怎样

**正确思维：**
- 关注当下正在发生什么
- 准备好各种可能的应对
- 让市场自己选择

### 情绪陷阱

**错误做法：**
- 因为恐惧在底部割肉
- 因为贪婪在顶部追高
- 因为侥幸不止损

**正确做法：**
- 按照系统信号操作
- 机械执行既定策略
- 无我无股票，只有走势

### 级别混乱

**错误操作：**
- 用日线级别期望做30分钟波段
- 看周线做日内交易
- 持仓时间与级别不匹配

**正确操作：**
- 级别清晰，操作一致
- 日线级别就拿日线周期
- 30分钟就做30分钟波段

## 金句精选

**关于市场：**
> 市场，就是要0投入去赚钱。

**关于操作：**
> 挣钱，本来就是很简单的事情，不过就是一个良好习惯与操作策略的结果，一点劲都不用费。

**关于心态：**
> 贪婪和恐惧，人的死穴，周末到了，请给自己一小时去找找自己！

**关于纪律：**
> 市场真正的成功，都是严格的操作下完成的。

**关于自我：**
> 人，总是自己骗自己；人，只能自己骗自己。没有你自己，谁都骗不到你。
