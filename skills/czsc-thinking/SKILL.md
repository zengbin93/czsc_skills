# 缠论思维

## 概述

本技能基于缠中说禅（2006-2008年）的原文思想，教导如何以缠论的方式思考和分析市场。缠论不仅是技术分析理论，更是一套完整的投资哲学和操作体系。

## 核心思维框架

### 三大核心理念

1. **不测而测** - 不预测市场未来，只分析当下正在发生什么
2. **完全分类** - 事先列出所有可能情况，为每种情况准备应对策略
3. **级别思维** - 任何分析都必须明确级别，不同级别有不同的操作策略

### 操作原则

- **生存第一**：永远不要让一次失败毁掉所有
- **机械操作**：严格按照系统操作，不加入个人情绪
- **节奏把握**：识别底、顶以及中间过程，在不同阶段采用不同策略

## 分析思维流程

当需要分析交易机会时，按以下顺序思考：

### 1. 明确级别
- 这是什么级别的机会？（周线？日线？30分钟？）
- 我的操作级别是什么？
- 级别是否匹配？

### 2. 识别结构
- 当前处于什么结构？（上涨？下跌？盘整？）
- 在结构的什么位置？（起点？中间？末端？）
- 中枢在哪里？

### 3. 判断买卖点
- 是第几类买卖点？（一买、二买、三买？）
- 确认条件是否满足？
- 有无背驰？

### 4. 评估风险收益
- 止损位在哪？
- 目标位在哪？
- 风险收益比如何？

### 5. 制定策略
- 买入/卖出的具体位置？
- 仓位多少？
- 后续如何应对各种情况？（完全分类）

## 实战思维模板

### 分析股票买卖点

**思维要点：**
- 先明确级别，再识别结构
- 找出当前的买卖点类型
- 评估可靠性和风险收益比
- 给出具体操作建议和止损位

### 分析大盘行情

**思维要点：**
- 纠正预测思维，分析当下结构
- 多级别共振分析（周线、日线、分时）
- 识别关键位置和多空力量对比
- 完全分类：列出所有可能及应对策略

### 制定交易策略

**思维要点：**
- 了解用户的级别和风险承受能力
- 基于当前结构给出具体点位
- 设定止损和目标价
- 说明各种情况的应对方案
- 强调纪律执行和风险控制

## 关键提醒

在所有分析中都要强调：

1. **永远强调级别** - 任何分析都要先明确级别
2. **不做预测** - 只分析当下，不预测未来
3. **完全分类** - 列出所有可能情况和应对
4. **纪律为王** - 强调严格执行系统信号
5. **风险第一** - 生存比盈利更重要
6. **避免情绪化** - 客观分析，机械操作
7. **反复强调** - 你不是多头也不是空头，你只是等待市场自己的选择

### Python 脚本工具

本技能提供了三个实用的 Python 脚本，帮助你快速上手使用 waditu/czsc 库进行缠论分析：

1. **fetch_market_data.py** - 通过 Tushare API 获取股票行情数据
2. **analyze_czsc_structure.py** - 使用 CZSC 对象分析K线数据，识别缠论结构
3. **signal_analysis.py** - 基于缠论结构生成买卖点信号

详细使用说明请参考：[scripts/README.md](scripts/README.md)

**快速开始：**

```bash
# 1. 获取市场数据
python scripts/fetch_market_data.py \
    --token YOUR_TUSHARE_TOKEN \
    --ts_code 000001.SZ \
    --start_date 20240101 \
    --end_date 20240614 \
    --output data.csv

# 2. 分析缠论结构
python scripts/analyze_czsc_structure.py \
    --input data.csv \
    --symbol 000001.SZ \
    --freq 日线

# 3. 生成买卖点信号
python scripts/signal_analysis.py \
    --input data.csv \
    --symbol 000001.SZ
```
